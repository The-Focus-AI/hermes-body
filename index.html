hermes-body

Reachy Mini front end for Nous Research's hermes-agent

What it is

OpenAI Realtime is the embodied "brain" — sub-second voice loop and robot motion tools. hermes-agent is the knowledge backend, reachable via a single ask_hermes tool. Cross-channel memory stays in sync via post-turn syncs.

Requires

  • An OpenAI API key (Realtime access)
  • A running hermes-agent Docker container with its OpenAI-compatible HTTP API enabled

See the GitHub repo for setup.

Tools

Robot-side: look, antennas, emotion, dance, camera, face_tracking, stop_moves, idle. Knowledge-side: ask_hermes.
